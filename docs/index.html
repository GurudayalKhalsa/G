This is the documentation for all classes and methods in G. Here you will find descriptions, properties, methods and example usage.

Core

Class

This is the base class for all Classes in G. All sub-classes of this inherit all methods from this. These child classes include G.Collection, G.Stage, G.Object and more.

Methods:
extend:

Creates a subclass of this class.

G.Class.extend(obj)

Parameters:

obj: an object literal containing the methods and properties associated with this class

Usage:

var Thing = G.Class.extend
({
  //constructor method
  initialize:function(foo)
  {
    this._foo = foo;

    //logs the parent class's initialize method (constructor) if exists
    console.log(this._super);
  },
  //example method
  getFoo:function()
  {
    return this._foo;
  }
});

var thing = new Thing("bar");
thing.getFoo(); //returns bar
Properties:
_super:

When inside of a method, if there exists a parent class and that parent class has this method, the this._super attribute references this method in the parent class.

Usage:

//call the parent method (this must be called within a method within a class)
this._super();

A Fast, Powerful, and Extendable 2D Game Framework

G is a javascript-based game and graphics framework for use with the HTML5 Canvas element.

Features:

  • Entity/Object Based
  • Full mobile support
  • Events
  • Mouse/touch/keyboard handling
  • Sound
  • Physics
  • Cameras
  • Multiple simultaneous stages
  • Collections
  • Multiple drawable shape objects
  • Images
  • Sprites
  • And Many More!
